PHP form class



This class generate the form elements.  Here is an example with single input field:

    2. 

  2.  session_start();
    3. 

  3.  include 'inc/Form.php';

  5.  $aCreate = array('name' => 'formAdd', 'action'=>'formAction.php', 'method'=>'post');
    6. 

  6.  echo $oFormHandler->form_start($aCreate);
    7. 

  7.  $aInput = array('name'=>'username', 'id'=>'username');
    8. 

  8.  $aInputValidation = array(
    9.

  11.  echo $oFormHandler->form_label('Username :', array('for'=>'username'));
